OOP Counter

State-Driven Documentation

Overview

The Counter Class is an Object-Oriented state management tool. It solves the problem of manual DOM manipulation by encapsulating logic (increment, decrement, and reset) within a single, reusable class.

By using class-based builds, we can spawn infinite independent counters on a single page without code duplications.

How To Use It

HTML Markup:

<div id="container" class="counter"></div>

Instantiation:

new Counter('#container', 10, 4);

Parameters: (targetID, startValue, stepValue)
