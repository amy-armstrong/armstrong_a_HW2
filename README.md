# Counter Component (JavaScript OOP – Homework #2)

This project builds on the counter we made in class and turns it into a small reusable component using JavaScript OOP.

---

## What It Does!!

The counter keeps track of a number and lets the user interact with it using buttons.

Features include:

- Increment button  
- Decrement button  
- Reset button  
- Custom starting value  
- Counter cannot go below `0`  
- Decrement and reset buttons become inactive when the count is `0`  
- Multiple counters can run on the same page

- JavaScript Modules
